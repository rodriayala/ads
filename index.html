Estructura de la Carrera Full Stack Javascript

Maquetación con HTML5
  • 2 Meses - 16 Clases

Introducción al HTML, en esta etapa vas a aprender todo el lenguaje de Maquetación con todas las etiquetas, hojas de estilo y las herramientas modernas de centrado. FlexBox y Bootstrap. Al finalizar esta etapa vas a poder hacer cualquier pagina web desde el maquetado.

Programación

Front End Javascript Básico
  • 2 Meses - 16 Clases

Aquí daras los primeros pasos como programador. Vas a aprender los conceptos básicos de Programación, tambien vas a trabajar con diferentes tipos de Variables, Modificar elementos del HTML con Javascript y Interactuar con el navegador y el usuario.

Back End NodeJS y Express
  • 2 Meses - 16 Clases

Programación en Javascript del lado del servidor. Aprenderas a desarrollar aplicaciones Web con Nodejs de la forma más eficiente y performante. Utilizando Express. Tambien podras crear API RESTFul, Manipular base de datos SQL y NoSql. Tanto MySQL como Mongo.

Front End Avanzado React
  • 2 Meses - 16 Clases

Vas a aprender a crear componentes, contenedores, jsx y redux, todas las herramientas necesarias para crear un front end de nivel profesional.

¿Quién puede sumarse a nuestra carrera de programación?

La carrera y todos sus cursos van dirigido a personas apasionadas por el mundo digital que quieran adquirir habilidades técnicas para trabajar en una empresa o que quieran llevar su proyecto personal o empresarial al próximo nivel. Durante el curso podrás trabajar con tu emprendimiento o con distintos proyectos reales que te brindaremos. Al ser un aula cooperativa, te verás beneficiado por las experiencias de tus compañeros.

  • Entusiastas, Docentes, Ingenieros, Informáticos, Profesionales de la Comunicación y el Diseño
  • Profesionales de la Administración, Profesionales del periodismo, la publicidad y el Marketing, Emprendedores con negocios que quieran llevar su idea al mundo, Personas apasionadas por aprender sobre el universo del desarrollo web

El curso no requiere de ningún conocimiento previo. Empezamos desde cero, con el cielo como límite.
